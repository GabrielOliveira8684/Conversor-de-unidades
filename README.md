# Conversor de Unidades (Python)

Um conversor interativo que fiz pra treinar manipulação de dados e estruturas de decisão em Python. É versátil, com várias unidades diferentes pra converter!

## O que ele faz?

O conversor te permite escolher entre 5 tipos diferentes de conversão:

- **Distância**: Metro, Quilômetro, Centímetro, Milímetro
- **Moeda**: Real (BRL), Dólar (USD), Euro (EUR)
- **Tempo**: Segundos, Minutos, Horas, Dias
- **Temperatura**: Celsius, Fahrenheit, Kelvin
- **Massa**: Gramas, Quilogramas, Libras, Onças

Depois é só escolher a unidade de origem, a de destino, colocar o valor e boom! A conversão sai na hora.

## Como testar

Se tu tiver o Python na máquina, é só mandar bala:

Clone o repo:
```bash
git clone https://github.com/GabrielOliveira8684/Conversor-de-unidades.git
```

Roda o script:
```bash
python conversor.py
```

Depois é só seguir as instruções na tela. Escolhe o tipo de conversão, as unidades e o valor!

## O que eu pratiquei aqui

- **Funções Reutilizáveis**: Criei uma função pra cada tipo de conversão (distância, moeda, etc).
- **Estruturas de Decisão**: if/elif/else pra validar entrada do usuário e fazer as conversões certas.
- **Validação de Entrada**: Loops enquanto a entrada não é válida, pra não deixar o programa quebrar.
- **Dicionários**: Usei pra armazenar as taxas de câmbio e nomes das unidades.
- **Formatação de Saída**: .4f e .2f pra deixar as respostas mais legíveis.
